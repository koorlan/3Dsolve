Description détaillée

Ce module, articulé autour de la structure Player permet de modéliser un joueur et de gérer l'interaction de celui-ci avec l'application.

Projet application - 3DSolve Version : 1.0

Programme pour la résolution et la manipulation virtuelle de casse-tête de type "Snake Cube"
