Compilation

Bibliothèque requises:

  • ftgl
  • freetype2
  • glfw3
  • glew
  • pour windows ( avec MinGW installez "pthreads-win32" package)

Linux

Note : libglfw2 just enough if package not present on repo otherwise goto glfw website, download and compile library. ``` $ sudo apt-get install ftgl-dev libftgl2 libfreetype6 libfreetype6-dev libglfw-dev liglfw3 libglew-dev libglew1.10 ```

Osx

Note : cake brew command ``` $ brew install ftgl freetype pkg-config glew homebrew/versions/glfw3 ```
